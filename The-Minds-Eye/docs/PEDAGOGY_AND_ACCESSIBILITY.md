# Pedagogy and Accessibility Framework

The Mind's Eye must be instructionally serious. The game mechanics should serve learning, not distract from it.

## Core Pedagogical Strategies

### 1. Inquiry-Based Learning

Students investigate a real historical problem using primary sources, maps, testimony, and evidence.

### 2. Place-Based Learning

The town itself becomes the learning environment. Students learn standards through local streets, buildings, people, businesses, and events.

### 3. Project-Based Learning

Longer missions can culminate in products such as:

- town council recommendation
- newspaper article
- court argument
- engineering plan
- historical marker proposal
- map annotation
- public exhibit

### 4. Retrieval Practice

Students repeatedly recall and apply knowledge during the simulation.

Examples:

- explain the source
- identify the claim
- cite the evidence
- connect the moment to the TEKS
- justify a team decision

### 5. Formative Assessment

The system collects evidence during the lesson instead of waiting until the end.

Evidence can include:

- short written responses
- team claims
- oral explanations
- evidence notebook entries
- teacher-marked observations
- exit tickets

### 6. Collaborative Learning

Students work in teams with structured roles. Each student still needs individual mastery evidence.

### 7. Differentiated Instruction

The same historical moment can be presented at different reading levels, language supports, response modes, and scaffolding levels.

## Mastery Model

The teacher sets the number of successful demonstrations required for each standard.

Example:

```text
TEKS 7.1(A)
Required successful demonstrations: 3
Student A:
- evidence notebook claim: successful
- team discussion justification: developing
- exit ticket: successful
- map explanation: successful
Status: mastery met
```

## Dice Mechanics

Dice rolls should create uncertainty and engagement. They should not replace academic evidence.

Dice can influence:

- whether a witness cooperates
- whether a team finds a clue quickly
- how much time remains
- whether a public argument persuades the crowd
- whether an investigation opens a bonus source
- whether an engineering/logistics plan succeeds

Dice should not determine whether a student knows the standard.

## Emergent Bilingual Supports

The system should support TEKS + ELPS alignment.

### Student Supports

- vocabulary coach
- cognates where appropriate
- word parts and roots
- sentence frames
- simplified syntax option
- side-by-side translation where appropriate
- read-aloud support
- visual glossary
- speaking rehearsal mode
- writing scaffold mode

### Word Coaching Example

```text
Word: livery
Student-friendly meaning: a place where people kept, rented, or hired horses and wagons.
Spanish support: caballeriza / establo de alquiler
Use it: The traveler stopped at the livery before going to the depot.
```

## 504 Supports

Possible teacher toggles:

- extended time
- reduced visual load
- chunked directions
- checklists
- text-to-speech
- speech-to-text
- alternate response mode
- reduced copying
- teacher notes panel
- focus mode
- movement break checkpoints

## SPED Supports

Possible teacher toggles:

- simplified reading passage
- guided notes
- fewer answer choices
- visual supports
- repeated directions
- role reassignment
- smaller task chunks
- alternate evidence submission
- explicit vocabulary preteach
- teacher-assisted response capture

## UDL Alignment

The platform should provide multiple means of:

- engagement
- representation
- action and expression

## Teacher Override

The AI can recommend mastery status, but the teacher must be able to override it.

The teacher remains the instructional authority.
