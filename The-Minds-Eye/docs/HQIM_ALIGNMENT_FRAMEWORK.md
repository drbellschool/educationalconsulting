# HQIM Alignment Framework

## Purpose

The Mind's Eye should be designed as an HQIM-ready platform, not just an engaging historical simulation.

HQIM means High-Quality Instructional Materials. The movement is national, though each state and district may define adoption requirements differently. The platform should therefore support broad HQIM principles while staying specifically aligned to Texas TEKS for the first build.

## Strategic Position

The Mind's Eye should be positioned as:

> An AI-assisted, source-grounded, standards-aligned instructional materials platform built around historical digital twins, authentic performance tasks, teacher approval, and measurable mastery evidence.

It should not be sold as a game first.

It should be designed as curriculum infrastructure that can generate, review, deliver, and measure high-quality learning experiences.

## HQIM Design Requirements

### 1. Standards Alignment

Every lesson, scenario, artifact, and mastery claim must connect to specific standards.

For Texas launch:

- TEKS
- TEKS Guide support where available
- STAAR-aligned writing expectations where appropriate
- ELPS supports for emergent bilingual students

### 2. Coherence

Lessons should not be disconnected one-off games.

The platform should support:

- unit sequences
- recurring skills
- cumulative evidence
- repeated mastery demonstrations
- prerequisite skill awareness
- spiraled vocabulary and concepts

### 3. Knowledge Building

The platform should build deep content knowledge through historical sources, maps, institutions, people, commerce, civic systems, science experiences, and writing artifacts.

Students should learn more than facts. They should build usable knowledge they can apply in reading, writing, discussion, investigation, and problem solving.

### 4. Cognitive Demand

Tasks must require meaningful thinking.

Avoid:

- shallow click-through play
- worksheet disguised as a game
- dice replacing reasoning
- generic AI writing prompts

Require:

- evidence use
- source comparison
- claim development
- mathematical reasoning
- scientific explanation
- civic decision-making
- written or oral justification

### 5. Teacher Usability

Teachers need materials they can actually use.

Each lesson package should include:

- overview
- TEKS alignment
- estimated time
- required materials
- source packet
- student-facing task
- teacher script or facilitation guide
- discussion questions
- artifact/rubric requirements
- accommodations
- mastery evidence plan
- answer/rationale guidance where appropriate

### 6. Assessment Coherence

Assessment should be embedded, not bolted on.

The platform should include:

- artifact requirements
- rubrics
- mastery attempts
- teacher scoring controls
- AI feedback recommendations
- revision cycles
- evidence dashboards

### 7. Equity and Access

Materials should support a wide range of learners.

Include:

- EB vocabulary coaching
- translation supports where appropriate
- sentence frames
- read-aloud options
- simplified text versions
- SPED and 504 accommodations
- alternate response modes
- accessible map views
- WCAG-aware design

### 8. Source Integrity

Historical accuracy is part of instructional quality.

Every visible historical claim should be:

- cited
- classified by provenance
- reviewable
- distinguished from simulation or creative licensing

## HQIM Evidence Binder

For district adoption, each unit or town package should be able to export an HQIM evidence binder.

Binder sections:

1. Standards alignment map
2. Unit/lesson sequence
3. Knowledge-building rationale
4. Source packet and citations
5. Student materials
6. Teacher facilitation guide
7. Rubrics and assessment artifacts
8. Accessibility and differentiation supports
9. Mastery evidence model
10. Review and provenance report
11. Data privacy and AI-use statement

## Product Rule

If The Mind's Eye wants district adoption, it must produce evidence that it is instructionally coherent, standards-aligned, teacher-usable, assessable, accessible, and historically trustworthy.

The simulation is the delivery method. The curriculum quality is the product.
