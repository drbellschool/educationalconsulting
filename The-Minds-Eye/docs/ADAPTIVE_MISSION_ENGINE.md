# Adaptive Mission Engine

## Purpose

The Adaptive Mission Engine coordinates individual, group, and whole-class missions while also supporting standards-based intervention.

It should help students practice deficits through meaningful town tasks rather than obvious remediation worksheets.

## Core Principle

Every intervention should feel like an opportunity to contribute, not a consequence of struggling.

The town should ask students for help in ways that naturally require them to practice the skill they need.

## Mission Scales

### Individual Missions

Assigned to one student.

Used for:

- individual mastery evidence
- targeted intervention
- extension
- writing practice
- vocabulary practice
- independent investigation

### Team Missions

Assigned to teams of 3-5 students.

Used for:

- collaboration
- role-based evidence gathering
- group decision-making
- lab/report tasks
- source comparison

### Whole-Class Missions

Assigned to the entire class.

Used for:

- major town events
- civic meetings
- railroad events
- public health issues
- trials
- elections
- disasters
- shared simulation days

## Adaptive Intervention Logic

The engine tracks:

- selected TEKS
- mastery attempts
- success rate
- misconceptions
- amount of support needed
- accommodations
- EB supports
- teacher notes
- preferred response mode

Then it assigns or suggests missions that create additional practice.

## Intervention Examples

### ELAR Evidence Deficit

If a student struggles to use textual evidence:

- newspaper editor asks them to verify a rumor
- judge requires a statement with evidence
- historian asks them to compare two accounts
- sheriff asks for proof before acting

### Math Deficit

If a student struggles with proportional reasoning:

- depot manager asks for freight cost calculation
- blacksmith asks for material estimates
- bridge crew asks for scale/distance calculation
- merchant asks for inventory comparison

### Science Deficit

If a student struggles with data interpretation:

- doctor asks them to compare water samples
- gardener asks which method performed best
- engineer asks for bridge-test results
- town council asks for a public health recommendation

## Difficulty Curve

Track independence, not just correctness.

Example:

| Attempt | Support Used | Result |
|---|---|---|
| 1 | Teacher prompt + vocabulary coach | Successful |
| 2 | Vocabulary coach only | Successful |
| 3 | Independent | Successful |

This helps teachers see whether mastery is durable and independent.

## Mission Validity Test

A mission is valid only if:

1. the town has a believable reason to ask for the work
2. the student must practice the target skill
3. evidence is captured
4. the teacher can review mastery
5. the student cannot complete the task without executing the TEKS

## Teacher Controls

Teachers can:

- accept/reject adaptive mission suggestions
- set intervention intensity
- assign extra practice privately
- assign missions to individuals, teams, or whole class
- prevent overloading a student
- override AI mastery conclusions
- turn off adaptive intervention during live events

## Student Experience

The student should experience intervention as meaningful contribution.

Bad:

> You failed evidence, so complete this practice.

Better:

> The Gazette editor heard you are careful with details. She needs you to check whether this rumor is supported before tomorrow's paper goes to print.

## Non-Negotiable

Adaptive missions must be dignifying, purposeful, and standards-aligned.

No student should be publicly marked as struggling through the game interface.
